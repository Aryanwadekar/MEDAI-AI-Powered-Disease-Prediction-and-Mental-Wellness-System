# MEDAI-AI-Powered-Disease-Prediction-and-Mental-Wellness-System
 MedAI is a smart healthcare application that predicts possible diseases based on user-input symptoms and helps users locate nearby hospitals for quick medical assistance.
Features
🔍 Disease Prediction
Enter symptoms and get probable diseases using machine learning algorithms.
📊 Data-Driven Analysis
Uses trained models on medical datasets to improve prediction accuracy.
📍 Nearby Hospital Locator
Suggests hospitals near the user for immediate consultation.
💻 User-Friendly Interface
Simple and clean UI for easy interaction.
⚡ Fast & Efficient
Provides quick results to assist in early decision-making.
🧠 Technologies Used
Machine Learning (for prediction)
Python
Flask / Django (if used backend – edit accordingly)
HTML, CSS, JavaScript (for frontend)
APIs for location services (Google Maps or similar)
🎯 Objective

To provide a smart assistant that helps users identify potential health issues early and guides them to the nearest healthcare facilities, reducing delays in treatment.
